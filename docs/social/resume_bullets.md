# Resume Bullets

- Built a Streamlit retail KPI dashboard that turns modeled operating data into KPIs, forecasts, scenarios, exports, and executive recommendations.
- Designed a SQLite-backed analytics workflow with deterministic sample data, reusable Python modules, and pytest coverage for KPI, forecast, scenario, recommendation, and export logic.
- Implemented explainable scenario modeling for marketing, pricing, cost, demand, and retention assumptions with risk classification and recommended actions.
- Created a rule-based recommendation engine that translates business metrics into prioritized actions with evidence and decision rationale.
- Developed an automated Playwright media workflow to generate portfolio screenshots, demo video, poster image, LinkedIn cover, and media manifest.
